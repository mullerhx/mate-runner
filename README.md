mate-runner
===========
Is a panel plugin for the MATE desktop environment that lets you run local commands on the fly.
Its not working yet, so dont hold your breath.



Dependencies
============
gtk+-2.0     >= 2.24
glib-2.0     >= 2.32
gio-2.0      >= 2.32
dbus-glib-1  >= 0.100.2
libsoup-2.4  >= 2.38
gio-unix-2.0 >= 2.32



Installing
==========
./autogen.sh
./configure
make && make install
